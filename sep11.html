SEP 11

Git and GitHub

In this unit of SEP, students will learn about the command line and how to use the command line instead of the mouse for traversing through files. Students will also learn about GitHub, a website where projects can be uploaded and other people can view and edit the code.

Web Design

WEb design. It has to do with webpages and it's a big unit, having three languages in it. Firstly, there's HTML. It determines what's in the page. Images, text, the boxes that those are in, and the images', text's, and the boxes' basic position on the page. Next, CSS gives the HTML style. That meaning that CSS tells the images, text, and boxes how to look and where to go on the page more specifically than HTML. This unit also has Bootstrap which is an extension to normal CSS; giving the page developer a lot more control over the page and its contents. Lastly, there's JavaScript. JS can edit things on a page after the page has been loaded depending on actions that are taking place on the page.
