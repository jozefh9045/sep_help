SEP 9

Scratch

The first unit in SEP is about a program called Scratch! It's sort of like a game where you connect blocks together to control what happens in the screen. It's a very simple introduction to simple concepts in coding like loops, where something is run over and over, and conditionals, where something will happen if a condition of something is met.

Robotics

Robotics! Like scratch, the lego robots are coded through a block-like system to teach some of the same basics as Scratch teaches; conditionals and loops. Robotics is different as you have an actual little robot to code to tell what to do!

Python

In the Python unit of 9th grade, students will start writing the code themselves instead of using pre-existing blocks. Like before, students will be focusing on conditionals and loops.
